## 目录说明
```jsx
 ├─app  //webApp（移动端）
 │  ├─public  //输出文件夹
 │  └─src  //源文件夹
 │      ├─assets  //资源
 │      │  ├─css 
 │      │  ├─images 
 │      │  └─js 
 │      ├─component //公共组件文件夹
 │      │  └─home
 │      └─page  //页面组件文件夹
 ├─img  //README所需图片文件夹
 │  ├─systems　//后台系统图片
 │  └─webApp  //webApp图片   
 ├─server   //服务器
 │  ├─data  //部分数据
 │  ├─db    // 数据库MongoDB
 │  ├─routers   //路由
 │  └─utils   //其他封装函数
 └─system   //后台（PC端）
    ├─public
    └─src
        ├─api   //封装的请求函数
        ├─assets
        │  ├─css
        │  └─images
        ├─component
        ├─data
        └─page
```
